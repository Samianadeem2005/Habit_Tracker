# HabitLoop : Minimalist Weekly Habit Tracker

A sleek, premium dark forest green single-page habit tracker built to help users build discipline, one day at a time. This app allows users to define daily habits, track them on a rolling weekly grid, navigate through historical data, and monitor active streaks.


## How to Run Locally

Since this project is built using a clean, vanilla frontend stack (HTML5, CSS3, and native JavaScript), it runs entirely in the browser without requiring any heavy installations or `npm install` commands.

### Option 1: Direct Run (Simplest)
1. Clone this repository to your local machine.
2. Navigate into the project directory.
3. Double-click the `index.html` file to open it instantly in any modern browser (Chrome, Edge, Firefox, Safari).

### Option 2: Live Server (Recommended for development)
If you are using VS Code:
1. Install the **Live Server** extension.
2. Right-click on `index.html` and select **"Open with Live Server"**.
3. The app will run locally.
